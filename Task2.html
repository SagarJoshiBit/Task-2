What is HTML?

HTML Stands for HyperText Markup Language, where HyperText stands for Link between web pages. Markup Language means Text between tags that define the structure.HTML is a markup language that is used to create web pages. It defines how the web page looks and how to display content with the help of elements. It forms or defines the structure of our Web Page, thus it forms or defines the structure of our Web Page. We must remember to save your file with .html extension.
